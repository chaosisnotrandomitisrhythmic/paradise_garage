```
┌───────────┐ ┌─────────┐ ┌────────┐ ┌─────────┐ ┌───────────┐
│ Technics L│ │ CDJ-L   │ │ Mixer  │ │ CDJ-R   │ │ Technics R│
│  453mm    │ │  329mm  │ │ 336mm  │ │  329mm  │ │  453mm    │
│  x 353mm  │ │  x 453mm│ │ x 410mm│ │  x 453mm│ │  x 353mm  │
│  h: 169mm │ │  h:118mm│ │ h:109mm│ │  h:118mm│ │  h: 169mm │
└───────────┘ └─────────┘ └────────┘ └─────────┘ └───────────┘
├─────────────────── ~2,020mm (with 30mm gaps) ──────────────────┤
```

| Measurement | Value |
|-------------|-------|
| Total minimum width | ~1,900mm |
| Recommended width (30mm gaps) | **~2,020mm (~2m)** |
| Depth | 453mm (CDJs are deepest) |
| Add behind for cables | 50–100mm |
| Height difference | Turntables 169mm vs CDJs/mixer 109–118mm |

**Note:** Turntables are ~60mm taller. Consider recessing them or raising CDJs/mixer for level playing surfaces.

## Desk Design Notes

- **Turntable isolation is critical** — decouple from bass vibration using rubber feet, isolation platforms, or a separate shelf
- **Standard DJ booth height** — 90–100cm (standing), 70–75cm (seated)
- **Cable routing** — plan channels underneath or behind for RCA + power
- **Ventilation** — CDJs and mixer need airflow, don't fully enclose the back
- **Material** — MDF or plywood (18mm+), avoid glass or metal (resonance)
